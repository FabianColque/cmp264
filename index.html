Visualization of Reviews Amazon
All over the world people are used to buy things on the internet y often need help to choose what they want to buy. So this file proposes to show a set of views that help the user to choose. With a database that are basically opinions on many products and are evaluated by sentiment analysis to be able to give a value on this review, The views shown in this file will show many reviews ways to get different results it will be very important for those you want to choose a product y its various details.
Matrix Review
We will use this view to display the detail of the features we can find in our produtos.Esta viewing opinions has two properties for each cell -Color: Determines the value of the feature if the polarity is redder is then more negative is greener and then is more positive. -Size Of the cell: This property determines the percentage amount between positive and negative for the same feature.

Time Visualization
